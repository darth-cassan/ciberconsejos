# CiberConsejos

Sitio web educativo (estático) con buenas prácticas de seguridad digital: recomendaciones para redes sociales, recursos oficiales y un generador de contraseñas.

Politécnico Jesús Marín.

## Características

- Secciones claras: redes sociales, recursos, ayuda y herramienta de contraseñas.
- Generador de contraseñas (con fortaleza, copiado y opciones).
- Diseño responsive y soporte de modo claro/oscuro.
- Interacciones accesibles: navegación por anclas, modales y tarjetas clicables.

## Tecnologías

- HTML + CSS + JavaScript (sin build step).
- Bootstrap 5 (layout y componentes).
- Font Awesome (iconos).
- Animate.css (animaciones puntuales).

## Notas

- Los centros y enlaces de ayuda están orientados a España (INCIBE/OSI/AEPD, 112, 091, 062, etc.).
- El contenido es informativo/educativo y no sustituye asesoramiento profesional.

## Contribución

- Issues y PRs son bienvenidos.
